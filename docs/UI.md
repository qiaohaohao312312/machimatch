# Machimatch — UI Design Manual

> *Feels like a letter from a friend who has lived everywhere.*

---

## 1. Design Philosophy

Machimatch should feel like it was made by hand, not generated by an algorithm.  
Every screen is a moment of intimacy—not a form to fill, but a conversation to have.

**Three words that guide every design decision:**
- **Handmade** — imperfect, warm, tactile
- **Curious** — open, exploratory, never clinical
- **Intimate** — like someone who knows you is helping you

**Reference feeling:**
- A psychogeographic map annotated by hand
- A traveler's journal with tickets and photos tucked in
- A hand-illustrated map where landmarks have personality
- Two people sitting by the Seine, no destination, just being

**Anti-references (what we are NOT):**
- Clean SaaS dashboards
- Real estate apps with cards and filters
- Anything that feels like a spreadsheet

## 2. Color Palette

### Primary Colors

| Name | Hex | Use |
|---|---|---|
| Parchment | `#F5EEE4` | Primary background — warm, aged paper feel |
| Ink | `#2C2825` | Primary text — not pure black, slightly warm |
| Teal Deep | `#1D5C6B` | Primary interactive — buttons, selected states, links |
| Teal Soft | `#3A7F8F` | Secondary interactive, hover states |

### Secondary Colors

| Name | Hex | Use |
|---|---|---|
| Sand | `#E8DDD0` | Cards, secondary backgrounds |
| Ochre | `#C4873A` | Accent, highlights, warm emphasis |
| Moss | `#6B7C5C` | Nature tags, secondary labels |
| Blush | `#D4A89A` | Soft accents, emotional moments |
| Route Red | `#C0392B` | Sparingly — map routes, key callouts only |

### Usage Rules
- Never use pure white (`#FFFFFF`) — always use Parchment or Sand
- Never use pure black (`#000000`) — always use Ink
- Teal Deep is the only "brand color" — use it intentionally
- Backgrounds should always feel like aged paper, never clinical white

## 3. Typography

### Typeface System

**Display / Headlines**
- Font: `Kaisei Tokumin` (Google Fonts)
- Feel: warm Japanese-origin serif, editorial, slightly literary
- Use: screen titles ("Where Will Your Next Home Be?"), story text

**Body / Questions**
- Font: `Lora` or `EB Garamond`
- Feel: readable, human, like a well-typeset book
- Use: question descriptions, neighborhood intros, body copy

**Handwritten Accent**
- Font: `Nanum Pen Script` (Google Fonts)
- Feel: personal, ink-pen handwriting, distinctly human
- Use: quiz answer options, floating examples in free-text screen, tags, small labels
- ⚠️ Use sparingly — only where it adds personality, not for readability

**UI Labels**
- Font: `Inter` or `DM Sans` (light weight)
- Feel: functional but not cold
- Use: navigation, small utility text, timestamps

### Type Scale (Mobile)

| Role | Size | Weight | Font |
|---|---|---|---|
| Screen title | 32–36px | Regular | Playfair Display |
| Section title | 22–24px | Regular | Lora |
| Body | 16px | Regular | Lora |
| Handwritten label | 16–18px | Regular | Caveat |
| UI label | 13–14px | Regular | DM Sans |

### Rules
- Headlines are never bold — weight comes from the serif, not boldness
- Line height is generous: 1.5–1.7 for body, 1.2–1.3 for display
- Never center-align body paragraphs — only titles and short labels

---

## 4. Spacing & Layout

- Base unit: 8px
- Screen padding: 24px horizontal
- Cards: 16px internal padding, 12px corner radius
- Generous white space — emptiness is intentional, not a mistake
- Content should never feel crammed

## 5. Components

### Frosted Glass (磨砂玻璃)
- Background: `rgba(255,255,255,0.18)` — translucent white over the parchment
- `backdrop-filter: blur(16px)` + `-webkit-backdrop-filter: blur(16px)`
- Top inner highlight: `box-shadow: inset 0 1.5px 0 rgba(255,255,255,0.55)` — simulates light catching the top edge of glass
- Border: `1px solid rgba(255,255,255,0.35)` — soft, not teal
- No solid background fill — the parchment texture must show through
- Feel: raised, glossy, like a frosted pane lifted off the page

### Quiz Cards (Multi-select)
- Rounded rectangle: `border-radius: 16px`
- Border: `1.5px solid #1D5C6B` (Teal Deep)
- Background unselected: `#F5EEE4` (Parchment)
- Background selected: `#1D5C6B` with white text
- Label: Caveat font, centered
- Slight shadow: `box-shadow: 2px 3px 0px #C4C0BA` — gives a paper-on-table feel

### Sliders (Quiet ↔ Lively)
- Track: `#D4CFC9` (muted sand)
- Active track: `#1D5C6B`
- Thumb: `#1D5C6B`, circle, `28px` diameter
- Labels: Caveat font at both ends

### Priority Scale (1–5)
- 5 stacked horizontal bars, full width
- Each bar: `height: 52px`, `border-radius: 4px`
- Unselected: `#1D5C6B` at 80% opacity
- Selected: `#1D5C6B` at 100% + slight scale(1.02)
- Label text inside each bar: DM Sans, white, left-aligned with 16px padding
- Examples: "1 — Not important at all" / "5 — Must be within 15 min"

### Free Text Input
- Border: `1.5px solid #1D5C6B`, rounded `24px`
- Background: transparent (shows Parchment background through)
- Floating examples: Caveat font, `#2C2825` at 45% opacity, scattered around the input (not in a list)
- The floating examples should feel like thoughts drifting past, not instructions

### Image Swipe (Window View)
- Full-bleed image with `border-radius: 16px`
- Option labels below image: Caveat font
- Selected option: underline in Teal Deep
- Unselected: `#2C2825` at 40% opacity
- Dot indicator: filled = Teal Deep, empty = `#C4C0BA`

### Neighborhood Story Card (Result Page)
- Full screen scroll, not a compact card
- Structure:
  1. Neighborhood name — Playfair Display, 28px
  2. "Your day here" label — Caveat, 14px, Ochre color
  3. Story text — Lora, 16px, generous line height
  4. Divider — hand-drawn style thin line (SVG)
  5. Tags — pill-shaped, Caveat font, Teal border
  6. Map — below the fold, supplementary

## 6. Illustration & Photography Style

### Photography
- Always lifestyle, never real estate
- Golden hour light preferred
- People present (from behind, or in context — not posed portraits)
- Locations: streets, cafés, riversides, markets, windows
- Never: empty apartments, white walls, floor plans

### Illustration
- Hand-drawn quality: think ink on paper, slight imperfection
- Map illustrations: named landmarks, dashed route lines, small icons
- Color: Parchment background, Ink lines, Teal and Ochre accents
- Can be used as: section dividers, loading states, empty states, onboarding

### Icons
- Line-only, not filled
- Slightly irregular stroke (not perfectly geometric)
- 1.5–2px stroke weight
- Same Ink color as text

---

## 7. Motion & Interaction

**Transitions**
- Page transitions: gentle fade + slight upward slide (200ms ease-out)
- Card selection: subtle scale 0.97 → 1.0 on tap (gives physical feedback)
- Slider: smooth drag, no snapping feel

**Loading States**
- Never a spinner — use a hand-illustrated animation or a slow-reveal text
- Story generation loading: display a poetic line like *"Finding your street..."* or *"Listening to the neighborhood..."* while generating

**Micro-interactions**
- Floating text on free-input page slowly drifts (very subtle, 0.03px/frame — barely perceptible)
- Story text reveals with a gentle typewriter-style fade-in, line by line

---

## 8. Voice & Tone (Copy Style)

The copy IS part of the UI. Every label, question, and placeholder is a design decision.

**Voice:** A well-traveled friend. Warm, curious, never condescending.

**Rules:**
- Questions are conversational, not survey-like
  - ✅ "What window would you like to wake up to?"
  - ❌ "Select your preferred view type"
- Use "you" and "your" everywhere
- Avoid: "optimize," "filter," "results," "data"
- Prefer: "find," "feel," "discover," "your day," "your street"
- Placeholder text should feel like whispered examples, not instructions
  - ✅ *"I have a cat..."*
  - ❌ *"Enter additional preferences"*
